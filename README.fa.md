# درآمدنامه · DaramadName

دفترِ درآمدِ محلی و گواهیِ درآمدِ قابلِ چاپ، برای فریلنسرها.

[English](./README.md) · [دموی زنده](https://sijav.github.io/daramad-name/) ·
[کتابخانه‌ی کامپوننت](https://sijav.github.io/daramad-name/storybook/)

فریلنسر «بی‌سند» است: نه فیش حقوقی دارد، نه گواهی درآمد. سفارت، صاحبخانه و حسابدار
سندی می‌خواهند که وجود ندارد. درآمدنامه آن را از روی دریافتی‌هایی که خودت ثبت
می‌کنی می‌سازد.

**داده‌ات هیچ‌وقت از مرورگرت بیرون نمی‌رود.** نه حساب. نه سرور. نه بک‌اند.

> وضعیت: **MVP** (`0.1.0-alpha.1`). همه‌ی سناریوهای اصلی پیاده‌سازی شده‌اند.

## ویژگی‌ها

- ⚡ دریافتیِ ارزی را در کمتر از ۱۵ ثانیه ثبت کن، با نرخِ تبدیل که همان لحظه‌ی ثبت
  فریز می‌شود.
- 📄 گواهیِ درآمدِ PDF دوزبانه (فارسی/انگلیسی) بساز.
- 📈 نمودارِ درآمدِ ۱۲ ماهه را ببین، با هشدار وقتی یک مشتری سهمِ زیادی از درآمدت است.
- 💾 محلی‌محور: داده‌ات هیچ‌وقت از مرورگر بیرون نمی‌رود، با پشتیبان و بازیابیِ
  JSON برای جابه‌جایی بین دستگاه‌ها.

## شروع سریع

```bash
npm install
npm run dev
```

سپس **http://localhost:5173** را باز کن.

فرمان‌های دیگر: `npm run build`، `npm test`، `npm run lint`، `npm run storybook`.
مجموعه‌ی استوری در کرومیومِ واقعی اجرا می‌شود، پس یک‌بار با
`npx playwright install chromium` نصبش کن.

## داده‌ی من کجاست؟

روی دستگاهِ خودت، در IndexedDBِ مرورگرت، و جای دیگری نه. بک‌اندی وجود ندارد، پس
چیزی آپلود، sync یا ردیابی نمی‌شود، و PDF هم داخلِ همان صفحه ساخته می‌شود.
جابه‌جایی بین دستگاه‌ها با فایلِ JSON است که از تنظیمات می‌گیری. بازیابی، آنچه را
هست **جایگزین** می‌کند نه ادغام، و پنجره‌ی ناشناس وقتی بسته شود همه‌چیز را فراموش
می‌کند، پس پشتیبان را جایی نگه دار که به آن اعتماد داری: تنها نسخه‌ی بیرون از
مرورگر است.

## تصمیم‌هایی که خوب است بدانی

- **معادلِ تومانیِ ثبت‌شده فریز می‌شود**، یک‌بار هنگامِ نوشتن حساب می‌شود و
  هیچ‌وقت دوباره نه، پس تغییرِ بعدیِ نرخ نمی‌تواند آنچه را واقعاً گرفتی بازنویسی
  کند.
- **میانگین ماهانه بر ماه‌های سپری‌شده تقسیم می‌شود، نه ماه‌های دارای درآمد**، چون
  سفارت این عدد را «ماهانه چقدر درمی‌آورد» می‌خواند.
- **دریافتیِ با تاریخِ گذشته، نرخِ همان تاریخ را می‌خواهد**، نه نرخِ امروز، و هشدار
  می‌دهد که مقدار فریز می‌شود.

## چطور ساخته شد

وایب‌کد شده، ولی ایجنت روی یک قانون‌نامه کار می‌کند، نه روی وایب. به مجموعه‌ای
ثابت از دستورها در [AGENTS.md](./AGENTS.md) پایبند است که ابزار اجرایشان می‌کند
نه حسنِ نیت: رشته‌ای که از بومی‌سازی جا بماند بیلد را می‌شکند، اگر ترجمه‌ای از
قلم بیفتد تستِ مستنداتِ استوری‌بوک رد می‌شود، تایپ‌ها بدون هیچ راهِ فرار بررسی
می‌شوند، و تا وقتی lint، تایپ‌چک، تست و بیلد همه سبز نشوند و تغییر در مرورگر در
حالِ اجرا دیده نشود، چیزی «تمام» به‌حساب نمی‌آید.

ساخته‌شده با React 19، TypeScript، MUI، Lingui، Dexie روی IndexedDB، Storybook و
Vitest.
