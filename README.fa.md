# درآمدنامه · DaramadName

دفترِ درآمدِ محلی و گواهیِ درآمدِ قابلِ چاپ، برای فریلنسر ایرانی.

[English](./README.md) · [دموی زنده](https://sijav.github.io/daramad-name/) ·
[کتابخانه‌ی کامپوننت](https://sijav.github.io/daramad-name/storybook/)

فریلنسر ایرانی «بی‌سند» است: نه فیش حقوقی دارد، نه گواهی درآمد. سفارت، صاحبخانه و
حسابدار سندی می‌خواهند که وجود ندارد. درآمدنامه آن را از روی دریافتی‌هایی که خودت
ثبت می‌کنی می‌سازد.

**داده‌ات هیچ‌وقت از مرورگرت بیرون نمی‌رود.** نه حساب، نه سرور، نه بک‌اند.

> وضعیت: **MVP** (`0.1.0-alpha.1`). همه‌ی سناریوهای زیر سرتاسری کار می‌کنند.

## چه می‌کند

- دریافتی را در حدود ۱۵ ثانیه ثبت کن؛ مبلغِ ارزی، نرخِ تبدیل را همان لحظه‌ی ثبت
  فریز می‌کند.
- دفتر را بر اساس مشتری و بازه‌ی تاریخ فیلتر و جمع کن.
- گواهیِ درآمدِ PDF دوزبانه (فارسی/انگلیسی) بساز.
- نمودارِ درآمدِ ۱۲ ماهه را ببین، با هشدار وقتی یک مشتری سهمِ زیادی از درآمدت است.
- به‌شکلِ فایلِ JSON پشتیبان بگیر و بازیابی کن تا بین دستگاه‌ها ببری.

## شروع سریع

```bash
npm install
npm run dev          # http://localhost:5173
```

`npm run build` · `npm test` · `npm run lint` · `npm run storybook`. مجموعه‌ی
استوری در کرومیومِ واقعی اجرا می‌شود، پس یک‌بار با
`npx playwright install chromium` نصبش کن.

## داده‌ی من کجاست؟

روی دستگاهِ خودت، در IndexedDBِ مرورگرت، و جای دیگری نه. بک‌اندی وجود ندارد، پس
چیزی آپلود، sync یا ردیابی نمی‌شود، و PDF هم داخلِ همان صفحه ساخته می‌شود.
جابه‌جایی بین دستگاه‌ها با فایلِ JSON است که از تنظیمات می‌گیری. بازیابی، آنچه را
هست **جایگزین** می‌کند نه ادغام، و پنجره‌ی ناشناس وقتی بسته شود همه‌چیز را فراموش
می‌کند، پس پشتیبان را جایی نگه دار که به آن اعتماد داری: تنها نسخه‌ی بیرون از
مرورگر است.

## تصمیم‌هایی که خوب است بدانی

- **معادلِ تومانیِ ثبت‌شده فریز می‌شود**، یک‌بار هنگامِ نوشتن حساب می‌شود و
  هیچ‌وقت دوباره نه، پس تغییرِ بعدیِ نرخ نمی‌تواند آنچه را واقعاً گرفتی بازنویسی
  کند.
- **میانگین ماهانه بر ماه‌های سپری‌شده تقسیم می‌شود، نه ماه‌های دارای درآمد**، چون
  سفارت این عدد را «ماهانه چقدر درمی‌آورد» می‌خواند.
- **دریافتیِ با تاریخِ گذشته، نرخِ همان تاریخ را می‌خواهد**، نه نرخِ امروز، و هشدار
  می‌دهد که مقدار فریز می‌شود.

بقیه‌ی این‌ها و استدلالِ پشتِ معماری در [AGENTS.md](./AGENTS.md) است.

## چطور ساخته شد

کد را Claude Code (ایجنتِ کدنویسیِ Anthropic) نوشته، سینا هدایتش کرده و گندم آن را
در Figma طراحی کرده، زیرِ یک قانون‌نامه‌ی نوشته‌شده که ابزار اجرایش می‌کند نه بر
پایه‌ی وایب. قوانین، چیدمانِ پوشه‌ها و تصمیم‌های اصلی در [AGENTS.md](./AGENTS.md)
است؛ هر خفه‌کردن و پینِ نسخه در [TECH-DEBT.md](./TECH-DEBT.md)، و آنچه عمداً بیرون
گذاشته شده در `PHASE-NEXT.md`.

استک: React 19، TypeScript، MUI، Lingui، Dexie روی IndexedDB، Storybook، Vitest.
